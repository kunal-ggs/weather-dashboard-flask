# Weather Dashboard

This is my first Flask project.

I created this Weather Dashboard using Python, Flask, HTML and CSS. It uses WeatherAPI to get real-time weather data.

Users can enter any city name and check:
- Temperature
- Humidity
- Weather Condition

## Technologies Used

- Python
- Flask
- HTML
- CSS
- WeatherAPI

## How to Run

1. Install required libraries:
   pip install flask requests

2. Run the project:
   python app.py

3. Open:
   (https://weather-dashboard-flask-5z63.onrender.com)
